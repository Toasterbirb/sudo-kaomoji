# sudo-kaomoji
Prints out cute sudo prompts

## Installation
```sh
make
make install
```

Example usage:
```
toasterbirb@tux ~ $ SUDO_PROMPT="$(sudo-kaomoji)" sudo su -
(o´ω`o)ﾉ Mayw I hav ur passwrd sir:
root ~ $
```

You can change the SUDO_PROMPT variable in your .bashrc to make the change permanent
